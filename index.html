.IFC

O formato .IFC e o conceito de interoperabilidade na indústria BIM (Building Information Modeling).

IFCjs

Biblioteca JavaScript open BIM para carregar e visualizar aquivos IFC no browser. Utiliza ThreeJs.

ThreeJs

Biblioteca JavaScript/API cross-browser usada para criar e mostrar gráficos 3D animados em um navegador web. Utiliza WebGL.
